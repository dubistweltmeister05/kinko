## ⚡ Power & Energy Display Validation Checklist

- [x] Active power direction (Forward / Backward) matches the actual current flow direction

- [x] PF label is properly aligned with the vertical column content and consistent with other parameters

- [x] Active power is displayed only as Forward / Backward (no + / − signs)

- [x] Individual Active Power screen indicates Forward / Backward direction consistently

- [x] Active, Reactive, and Apparent parameter screens are aligned:
  - [x] Consistent parameter naming
  - [x] Consistent horizontal value formatting
  - [x] Layout matches the Apparent parameter screen

- [x] Active energy import and export symbols align with correct quadrant operation
  - [x] Import and export are displayed in opposite directions

- [x] Active energy export is displayed as Forward / Backward (no + / − signs)

- [x] Reactive energy is displayed as Capacitive / Inductive (no + / − signs)
